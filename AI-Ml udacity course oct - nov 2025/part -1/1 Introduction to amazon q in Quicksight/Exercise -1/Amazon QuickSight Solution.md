## Introduction to Amazon QuickSight with Roller Skate Sales Dataset

Explore how to use Amazon QuickSight for business data analysis using a roller skating equipment sales dataset. This session focuses on creating and managing datasets for visualization.

### Steps:

-   **Login and Setup**: Access Amazon QuickSight and navigate to the datasets section.
-   **Create a New Dataset**:
    -   Click the "New Dataset" button.
    -   Upload a custom file, "RollerSkateSales.csv".
    -   Edit dataset settings, preview, and prepare data.
-   **Modify Dataset Fields**:
    -   Rename fields, e.g., "Name" to "Customer Name".
    -   Change "Currency" to "Purchase Amount" and "Color" to "Product Color".
    -   Set up a calculated field "High-Value Customer" using a purchase amount threshold.
-   **Publish Dataset**: Save and make the dataset available for future use.

### Analysis and Visualization:

-   **Create New Analysis**:
    -   Start a new analysis with the Roller Skate Sales dataset.
    -   Choose a vertical bar chart visual type.
    -   Set parameters like x-axis, value, and color group.

### Dashboard Creation:

-   **Design and Publish Dashboard**:
    -   Build and name the dashboard "Roller Skate Sales Dashboard".
    -   Use it to showcase visualizations and insights from the data.
