
## Refining Dashboards with Amazon Q in QuickSight

Learn to use Amazon QuickSight and its generative Q&A experience to refine an existing dashboard using a dataset focused on clinical trials.

### Dataset Preparation:

-   Open QuickSight and go to the Topics area.
-   Create a new Sample Topic using the Clinical Trials dataset.
-   Ensure the Q&A experience is enabled before creation.
-   Once prepared, access the Datasets section to find and select "Q-pharmaceutical and clinical trials."

### Building Visuals:

-   Use "Build visual with Q" to add charts.
-   Visualize the number of test subjects by state and the average age of participants by study group.
    -   Update the chart type using "Edit with Q" and change visuals to a vertical bar chart.
    -   Format titles for clarity (e.g., "Subject States of Residence").
    -   Adjust sort options for data presentation, ensuring proper order.

### Dashboard Publishing:

-   Finalize and publish your analysis as "Drug Trial Dashboard."
-   Enable data Q&A for dynamic insights.
-   Publish the newly created dashboard.
