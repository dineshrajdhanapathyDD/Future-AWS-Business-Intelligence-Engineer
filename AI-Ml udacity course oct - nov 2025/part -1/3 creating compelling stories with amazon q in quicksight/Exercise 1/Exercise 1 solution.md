## Using Amazon Q in QuickSight for Effective Data Storytelling

Learn how Amazon Q can facilitate storytelling in data analysis.

### Steps to Build a Data Story:

1.  **Open a Dashboard:**
    -   View visuals of recent roller skate sales.
2.  **Create a Data Story:**
    -   Click the "build with Q" button.
    -   Add a prompt focusing on sales influence by customizability.
3.  **Choose Visuals:**
    -   Select "number of customizations by model," "sales by model," and "sales by color."
4.  **Build and Modify the Story:**
    -   Use different styles and review the initial version.
    -   Update the description to include country influences alongside customization and color.
5.  **Include Additional Visuals:**
    -   Add "sales by country" for a complete overview.

### Understanding the Impact:

Enhance storytelling with insights on customizability and geographical sales contributions, forming an engaging narrative that highlights data significance.
