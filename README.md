## Course 3


### Building Amazon Q Apps using Amazon Q Business

This course, "Building Generative AI Apps with Amazon Q Business," guides participants through the creation of powerful generative AI applications using Amazon Q Apps. Beginning with an introduction to the unique features of Amazon Q Apps, learners will explore methods for developing reusable app components. The course emphasizes best practices for sharing and integrating these applications into various workflows. A key project involves designing an AI-powered career coach assistance app, which utilizes AI to enhance the career coaching process through tailored recommendations and insights. By the end of the course, participants will gain hands-on experience and practical skills in developing impactful AI-driven solutions.
